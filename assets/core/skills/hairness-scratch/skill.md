Use explicit, lightweight working memory.

- Without an explicit request to create or join a Scratch, write nothing.
- A Scratch is `.overlay/scratches/<slug>/scratch.md`; it has no manifest,
  session journal, status machine or active pointer.
- Create a short title and only the minimum initial context the user accepted.
- Update it at meaningful boundaries: accepted decisions, changed constraints,
  handoff context or next steps.
- Never store a transcript or hidden reasoning.
