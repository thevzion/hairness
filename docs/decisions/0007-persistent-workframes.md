# 0007: Persistent Workframes

Workframes stores missions, segments, frames, boundaries, and semantic events without storing conversation. Closed segments remain immutable and require a SegmentDigest.

Workframes preserves trajectory; specialized artifacts preserve meaning; live sources prove current truth.
