# Agentic assets

An agentic asset is a versioned unit that changes what an agent can understand or do. It is broader than its transport:

| Transport term | Asset meaning |
| --- | --- |
| prompt or instruction | cognitive policy |
| skill or command | human-invokable operation |
| hook | lifecycle contribution |
| CLI or MCP | source or effect transport |
| worker profile | bounded execution role |
| schema or gate | verifiable contract |
| method | reusable reasoning procedure |

Treating these as software means they have owners, versions, inputs, results, tests, provenance, compatibility, and release gates. Hairness provides that foundation without claiming a universal asset registry in protocol `0.2`.

Agentic capital is the strategic accumulation of useful, maintained agentic assets. It is a product concept, not a protocol type.
